# Claudia

Assistente pessoal inteligente com voz, IA e calendário.

## Deploy no Vercel

1. Crie um repositório no GitHub e faça upload desta pasta
2. No Vercel, importe o repositório
3. Framework Preset: **Other**
4. Clique em Deploy

## Como usar

- **Agenda**: clique em qualquer dia para ver os eventos
- **Voz**: clique no microfone 🎙️ e fale seu compromisso
- **Texto**: escreva no campo e pressione Enter
- **Secretária**: aba de chat para perguntar sobre sua agenda
- **Intenções**: planos sem data definida
